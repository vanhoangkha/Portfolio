# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [2.0.0] - 2025-11-14

### Added
- React + TypeScript implementation
- Internationalization (i18n) support for English and Vietnamese
- Blog system with filtering and search
- Certifications section with modal details
- Testimonials carousel
- Admin dashboard with analytics
- Newsletter subscription
- Comment system for blog posts
- Project filtering by technology
- SEO optimization with structured data
- Responsive design improvements
- Dark/Light theme toggle
- PWA support

### Changed
- Migrated from vanilla JavaScript to React
- Updated UI with modern design patterns
- Enhanced performance with code splitting
- Improved accessibility compliance

### Security
- Added CSRF protection
- Implemented reCAPTCHA integration
- Secure authentication for admin panel

## [1.0.0] - 2024-01-01

### Added
- Initial portfolio website
- HTML5, CSS3, Vanilla JavaScript
- Responsive design
- Basic sections: Hero, About, Experience, Skills, Projects, Contact
- Service Worker for offline support
- SEO optimization
